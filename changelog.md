Added two new keys for the `bmInput` tag. `placeholder` and `selected`.
Basically it's a shortcut for allowing you to set a tag without any optional modifiers.

```js
...
{
  title: 'Price',
  type: 'bmInput',
  selected: '$129.99',
  placeholder: 'e.g. $99.99'
}
...
```
